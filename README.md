# PyTemplates
Provide developers with python templates so that they can easily develop python applications.

- [CommandApp](CommandApp) - The Python [CommandApp.py](CommandApp/CommandApp.py) is a template of command tools with arguments.
- [PlayImages](PlayImages) - The Python [PlayImages.py](PlayImages/PlayImages.py) plays images or video optionally step by step. The app is also a template. We can modify it to develop applications of computer vision.
- [Setup](Setup) - It provides templates about how to use setup.py to install your package in python environment.
